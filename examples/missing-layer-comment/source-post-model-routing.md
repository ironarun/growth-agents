# Sample Source Post: Model Routing

AI costs are rising as enterprises move more work into production AI systems.

One practical response is model routing: send simple work to cheaper models and reserve frontier models for complex or high-value tasks.

That is useful cost infrastructure because not every task needs the most expensive model.

Model routing can help teams control spend, reduce latency, and avoid wasting frontier-model calls on routine work.

But routing decides which model should answer.

It does not decide whether the answer is defensible enough to act on.

A perfectly routed answer can still be incomplete, overconfident, or wrong.
