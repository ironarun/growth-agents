## Operating Boundary

The operator should prefer small, reviewable workflow steps over broad autonomy.

It must stop and report when a command fails, when secrets are missing, when scope is unclear, or when a requested action would post publicly, spend money, upload ads, scrape restricted platforms, or modify production systems without explicit approval.
